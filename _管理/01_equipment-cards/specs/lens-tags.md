# Step 14 規格：鏡頭功能標籤

## 功能描述

為鏡頭類料卡加入方位/功能標籤（前鏡、後鏡、室內鏡、特殊鏡頭），讓使用者可以搜尋和篩選。

## 設計決策

使用現有 `tags TEXT[]` 欄位，不新增欄位。

標準功能標籤：
- `前鏡`、`後鏡`、`左鏡`、`右鏡`
- `室內鏡`
- `廣角特殊`

## 執行方式

### 批次打標（初始化）
在 `_開發檔案/scripts/` 新增 `tag-lens-cameras.js`，根據品名關鍵字自動打標：
- 品名含「前」→ 加 `前鏡` tag
- 品名含「後」→ 加 `後鏡` tag
- 品名含「室內」→ 加 `室內鏡` tag
- 其餘 category=鏡頭 且無方位標籤 → 加 `廣角特殊`（人工確認後再決定）

### 編輯 UI 建議值
`CardFormDialog` 的 tags 輸入區加入預定義建議清單（`<datalist>` 或 chip-picker），包含所有標準功能標籤。管理員可選擇或自由輸入。

## 驗收標準

- category=鏡頭 的料卡在 tags 中有對應方位標籤
- 搜尋「室內鏡」能命中對應料卡
- 編輯表單有鏡頭功能標籤的建議選項
