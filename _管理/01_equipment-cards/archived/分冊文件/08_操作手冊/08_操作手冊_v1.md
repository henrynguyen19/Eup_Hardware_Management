# 設備料卡系統 操作手冊

> **版本說明**：v2（2026-05-03）新增 Google OAuth 登入說明與管理員操作章節。

---

## 第 1 章　這個系統是什麼？

### 1-1 用來解決什麼問題

以前查一個設備，要打開 Excel 翻來翻去，照片也是散在不同資料夾，很花時間。

**設備料卡系統**把所有設備的照片和資訊集中在一個網頁上，用手機或電腦打開就能直接看，不需要安裝任何程式。

### 1-2 裡面有什麼

| 項目 | 數量 |
|------|------|
| 設備料卡（筆） | 786 筆 |
| 主照片（張） | 786 張 |
| 細節照片（張） | 1,091 張 |

每筆料卡包含：
- 設備名稱、廠商、料號
- 分類（主機／鏡頭／螢幕／線材…）
- 標籤（如 4G、RFID、HS昇銳）
- 備註說明
- 主照片 ＋ 多張細節照片

### 1-3 能做什麼

| 功能 | 說明 |
|------|------|
| 📷 照片牆 | 一次看所有設備的縮圖 |
| 🔍 模糊搜尋 | 輸入關鍵字，相關設備馬上出現 |
| 🗂️ 篩選排序 | 按分類、狀態篩選 |
| 🖼️ 細節照片 | 點進去看大圖和多角度細節 |
| ✏️ 管理後台 | 管理員可新增、編輯、刪除料卡（含照片）|

### 1-4 系統網址

```
https://equipment-cards.vercel.app
```

## 第 2 章　登入系統

### 2-1 登入前的準備

你需要一個 **公司 Google 帳號**（`@eup.com.tw`），這是系統的唯一登入方式。

> 💡 不需要記帳號密碼，直接用 Google 帳號一鍵登入。

### 2-2 登入步驟

1. 用瀏覽器（建議 Chrome 或 Edge）開啟：
   ```
   https://equipment-cards.vercel.app
   ```

2. 系統偵測到你尚未登入，會自動跳到登入頁面：

   ```
   ┌─────────────────────────────────────┐
   │        設備料卡管理系統              │
   │     請使用公司 Google 帳號登入       │
   │                                     │
   │  [G]  使用 Google 帳號登入           │
   └─────────────────────────────────────┘
   ```

3. 點「**使用 Google 帳號登入**」按鈕

4. 瀏覽器會彈出 Google 登入視窗（或直接跳轉）：
   - 選擇你的 **公司帳號**（`xxx@eup.com.tw`）
   - 如果你已登入多個帳號，請確認選的是公司帳號，**不是個人 Gmail**

5. Google 驗證完成後，自動跳回系統首頁，看到設備照片牆代表登入成功

### 2-3 登出

畫面右上角會顯示你的 email 和「**登出**」按鈕。點登出後會回到登入頁。

### 2-4 我的帳號是管理員嗎？

右上角 email 旁邊若顯示「**管理員**」藍色標籤，代表你有管理員權限。
沒有顯示，代表是一般使用者（只能瀏覽）。

## 第 3 章　瀏覽設備料卡

### 3-1 照片牆（首頁）

登入後看到的主畫面是一個**卡片網格**，每張卡片代表一個設備：

```
┌─────────┐ ┌─────────┐ ┌─────────┐
│  [照片] │ │  [照片] │ │  [照片] │
│         │ │         │ │         │
│ 1000003 │ │ 1000005 │ │ 1000007 │
│衛星定位器│ │衛星定位器│ │貨物定位器│
│ 格瑪車機 │ │ 格瑪車機 │ │         │
└─────────┘ └─────────┘ └─────────┘
```

**卡片上顯示的資訊：**
- 右上角小數字（如 `+3`）：代表這個設備有 3 張細節照片
- 右下角紅底「停產」標籤：代表該設備已停產
- 廠商名稱（如「格瑪車機」）

### 3-2 卡片數量

搜尋列下方會顯示目前符合條件的結果數量，例如：
```
顯示 786 / 786 筆
```

## 第 4 章　搜尋與篩選

### 4-1 模糊搜尋

畫面最上方有一個搜尋框：

```
[ 🔍 搜尋料號、品名、廠商、備註…（支援模糊比對）]
```

**可以搜尋的欄位：**

| 搜尋內容 | 範例 |
|----------|------|
| 料號 | 輸入 `1000003` |
| 品名 | 輸入 `DVR` 或 `定位器` |
| 廠商 | 輸入 `格瑪` 或 `昇銳` |
| 標籤 | 輸入 `RFID` 或 `4G` |
| 備註 | 輸入備註裡的關鍵字 |

**模糊比對**：不需要輸入完整名稱，輸入一部分也能找到。  
例如輸入「衛星」可以找到所有含「衛星定位器」的設備。

**清除搜尋**：點搜尋框右側的 **×** 圖示。

---

### 4-2 分類篩選

搜尋框下方是**分類按鈕列**：

```
[全部] [主機] [鏡頭] [螢幕] [天線] [儲存媒體] [線材] [配件] [耗材] [工具] [國外設備]
```

點任一分類，畫面只顯示該分類的設備。點「**全部**」回到全部顯示。

---

### 4-3 狀態篩選

分類按鈕列的後半段：

```
[全部狀態] [現役] [停產]
```

- **全部狀態**：顯示所有設備（預設）
- **現役**：只顯示目前在用的設備
- **停產**：只顯示已停產的設備

---

### 4-4 排序

搜尋框右側有排序下拉選單：

```
[ ↕ 料號排序 ▼ ]
```

- **料號排序**：依料號數字排列（預設）
- **品名排序**：依品名中文字母排列

> 💡 有輸入搜尋關鍵字時，排序改由「相關度」決定（最相關的排在前面），手動排序不生效。

---

### 4-5 清除所有篩選

若有篩選條件啟用，篩選列最後會出現「**× 清除篩選**」按鈕，點一下清除所有條件。

## 第 5 章　查看設備詳情

### 5-1 開啟詳情

**點擊任何一張卡片**，即可開啟該設備的詳情畫面（Lightbox）。

### 5-2 詳情畫面說明

```
┌──────────────────────────────────────────┐
│                                    [✕]   │
│                                          │
│              [主照片大圖]                 │
│                                          │
│   ← [縮圖1][縮圖2][縮圖3] →              │
│                                          │
│   料號：1000003                          │
│   品名：S168-3G衛星定位器                │
│   分類：主機    廠商：格瑪車機            │
│   狀態：現役                             │
│   標籤：[格瑪車機] [3G]                  │
│   備註：（若有備註會顯示在此）            │
└──────────────────────────────────────────┘
```

### 5-3 切換細節照片

- 點擊下方**縮圖列**：切換到該張照片
- 點大圖可**放大**檢視
- 按鍵盤 **← →** 或點兩側箭頭：切換上一張 / 下一張
- 按 **Esc** 或點右上角 **✕**：關閉

## 第 6 章　管理員操作

> 本章內容僅適用於**管理員帳號**。一般使用者不會看到以下功能。

---

### 6-1 新增料卡

**步驟：**

1. 畫面**右下角**找到「**＋ 新增料卡**」藍色圓角按鈕，點擊

2. 「新增料卡」表單開啟：

```
┌──────────────────────────────────────┐
│ 新增料卡                         [✕] │
├──────────────────────────────────────┤
│ 料號 *    [_______________]          │
│ 品名 *    [_______________]          │
│ 分類      [— 未分類 —  ▼]           │
│ 狀態      [現役        ▼]           │
│ 廠商      [_______________]          │
│ 標籤      [_______________]          │
│           （逗號分隔）               │
│ 備註      [_______________]          │
│                                      │
│ 主照片    [↑ 上傳主照片]             │
│ 細節照片  [縮圖...] [+]              │
│                                      │
├──────────────────────────────────────┤
│ [取消]                     [建立]    │
└──────────────────────────────────────┘
```

3. **填寫各欄位**（詳見 6-4）

4. 點「**建立**」完成

---

### 6-2 編輯料卡

**步驟：**

1. 將滑鼠移到要編輯的**卡片上方**

2. 左上角出現兩個小按鈕：
   ```
   [✏️ 編輯]  [🗑️ 刪除]
   ```

3. 點「**✏️ 編輯**」按鈕

4. 「編輯料卡」表單開啟（欄位已預填目前資料）

5. 修改需要更改的欄位

6. 點「**儲存**」完成

> 💡 料號（equipment_id）無法修改。若料號輸入錯誤，需要刪除後重新建立。

---

### 6-3 刪除料卡

**步驟：**

1. 滑鼠移到要刪除的卡片上方，點左上角「**🗑️ 刪除**」按鈕

2. 出現確認提示：
   ```
   確定要刪除「S168-3G衛星定位器」？
   此操作無法還原，Cloudinary 照片也會一併刪除。
   ```

3. 點「**確定**」執行刪除

> ⚠️ 刪除後**無法還原**，連同 Cloudinary 上的主照片和所有細節照片都會一起刪除。請確認後再執行。

---

### 6-4 表單欄位說明

| 欄位 | 必填 | 說明 |
|------|------|------|
| 料號 | ✅ | 唯一識別碼，建立後不可修改。格式如 `1000003` |
| 品名 | ✅ | 設備的完整名稱，例如「S168-4G衛星定位器」 |
| 分類 | — | 從下拉選單選擇（主機、鏡頭、螢幕…等 10 類）|
| 狀態 | — | 現役（正在使用）或停產（已停止使用）|
| 廠商 | — | 供應商或品牌名稱，例如「格瑪車機」|
| 標籤 | — | 搜尋用的關鍵字，多個標籤用逗號分隔：`HS昇銳, RFID, 4G` |
| 備註 | — | 補充說明，搜尋時也會比對備註內容 |

---

### 6-5 照片管理

#### 上傳主照片

在新增或編輯表單中：

1. 點「**↑ 上傳主照片**」按鈕
2. 選擇電腦上的圖片檔案（JPG、PNG 皆可）
3. 照片縮圖出現代表選取成功（尚未上傳）
4. 點「**建立**」或「**儲存**」時才正式上傳

**更換主照片（編輯模式）：**
- 在現有縮圖旁點「**更換照片**」直接上傳新照片（立即生效）
- 點「**刪除照片**」移除目前主照片（立即生效）

#### 新增細節照片

在編輯模式下：

1. 細節照片區點「**+**」按鈕
2. 選擇一張或多張圖片（可多選）
3. 照片立即上傳並顯示在細節照片列

#### 刪除細節照片

滑鼠移到細節照片縮圖上，出現垃圾桶圖示，點擊即可刪除（立即生效）。

> 💡 **新增與刪除細節照片**都是即時生效，不需要點「儲存」。
> **文字欄位**修改後需要點「儲存」才會存檔。

## 第 7 章　常見問題

**Q：用手機可以嗎？**  
A：可以，直接用手機瀏覽器開啟即可，版面會自動適應螢幕大小。

**Q：照片載入很慢怎麼辦？**  
A：照片存放在雲端，第一次載入稍慢，之後瀏覽器會暫存所以會更快。若一直很慢，請確認網路連線正常。

**Q：找不到某個設備怎麼辦？**  
A：試著只輸入關鍵字的一部分（例如找「FUHO 4G DVR」，試著輸入「FUHO」或「DVR」）。若確實找不到，可能是尚未建入系統，請聯絡管理員。

**Q：想要用個人 Gmail 登入可以嗎？**  
A：不行，系統限定公司 `@eup.com.tw` 帳號登入。

**Q：登入後出現「此 Google 帳號未被授權」怎麼辦？**  
A：代表你用的帳號不是 `@eup.com.tw`。請重新登入並選擇正確的公司帳號。

**Q：系統無法開啟怎麼辦？**  
A：請確認：（1）網路正常、（2）用 Chrome 或 Edge 開啟、（3）清除瀏覽器快取後重試。若仍無法，請告知管理員。

**Q：帳號可以讓別人使用嗎？**  
A：請勿共用帳號，每人使用自己的公司 Google 帳號登入。

**Q：發現料卡資料有誤怎麼辦？**  
A：請聯絡管理員修正，或若你是管理員，直接點卡片上的編輯按鈕修改。
