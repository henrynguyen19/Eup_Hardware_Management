# 設備料卡管理系統 — 專案概覽

## 一句話描述

取代 Excel 的部門設備照片牆與管理後台，供 10–50 人內部使用，786 筆料卡。

## 技術架構

- **前端**：Next.js 14 App Router + Tailwind CSS + shadcn/ui
- **資料庫/Auth**：Supabase（PostgreSQL + Google OAuth）
- **照片儲存**：Cloudinary（25GB 免費）
- **部署**：Vercel Hobby（GitHub push 自動部署）
- **搜尋**：Fuse.js（前端全客戶端）

## 已知技術債 / 注意事項

- `equipment_id` 為 TEXT PRIMARY KEY，更新料號時需特別處理（API 已支援）
- Fuse.js 純數字查詢走精確 `includes` 不走模糊算法
- 照片編輯採暫存機制（按儲存才實際呼叫 Cloudinary API）
- shadcn/ui Popover 在 overflow:hidden 父容器需改用 fixed 定位
- `useSearchParams()` 必須包在 `<Suspense>` 內
