## 使用說明

## [2026-05-23] P01 | 技術選型回顧（遷移前存檔）

**討論背景**：從舊版 CLAUDE.md 結構遷移至 D 型架構，順便存檔原有技術選型決策。

**決策結果（已確認）**：
- 前端：Next.js 14 App Router + Tailwind CSS + shadcn/ui
- 資料庫：Supabase（PostgreSQL + Auth + RLS）
- 照片儲存：Cloudinary（25GB 免費）
- 部署：Vercel Hobby（push 即自動部署）
- 前端搜尋：Fuse.js（786 筆小資料，全客戶端）

**排除原因**：
- Cloudflare R2：只有 10GB 免費，且需信用卡
- Firebase：比 Supabase 複雜，PostgreSQL 更熟悉
- 伺服器端搜尋：資料量小，Fuse.js 即可，省 API 費用

**影響的步驟**：所有 Step 1–8（已完成）
